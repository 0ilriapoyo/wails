# Screen

These methods provide information about the currently connected screens.

### ScreenGetAll

Returns a list of currently connected screens.

Go: `ScreenGetAll(ctx context.Context) []screen`<br/>
JS: `ScreenGetAll()`


#### Screen

```go
type Screen struct {
	IsCurrent bool
	IsPrimary bool
	Width     int
	Height    int
}
```

```ts
interface Screen {
    isCurrent: boolean;
    isPrimary: boolean;
    width : number
    height : number
}
```
